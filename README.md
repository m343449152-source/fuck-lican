这是一个通过自己动手，利用合法手段从老赖手中拿回自己财产的小教程。

因为实在非常抱歉，在多名群友向公安部门进行刑事控告后，均以“不由本部门管辖”的理由退回，本人也以信访等方式与公安部门进行一个月拉扯后，不得不放弃，并以民事起诉的方式来维权。所以在目前的法治进程下，我们无法将坏人绳之以法，无法对坏人进行任何惩罚，最多只能拿回本就属于自己的财产。同时也无法通过我的努力帮助大家要回钱，只能大家亲力亲为，自己动手丰衣足食。
